# entornosDesarrollo

Repositorio con ejercicios de **PHP** realizados durante el módulo de **Desarrollo de Aplicaciones Web (DAW)** en **STUCOM**. Incluye prácticas y actividades de clase para reforzar los conceptos básicos y avanzados del lenguaje PHP.

## Contenido

Este repositorio contiene carpetas y archivos con ejercicios organizados por temas o entregas:

- **Ejercicios/** – Primer conjunto de ejercicios de PHP.
- **Ejercicios2/** – Segundo conjunto de ejercicios.
- **Ejercicios3/** – Tercer conjunto de ejercicios.
- **PRACTICASCLASE/** – Prácticas realizadas en clase.
- Archivos sueltos (`Ejercicio2.php`, etc.) con ejemplos y pequeñas prácticas. :contentReference[oaicite:0]{index=0}

## Objetivo

El objetivo de este repositorio es:

- Guardar de forma ordenada y versionada los ejercicios que se realizan en clase y en casa.
- Facilitar el repaso y la mejora continua de las prácticas de PHP.
- Documentar brevemente la lógica y el propósito de cada ejercicio (cuando sea aplicable).

## Tecnologías y Requisitos

Los ejercicios están escritos en **PHP**. Para ejecutar los scripts se recomienda:

1. Tener instalado un servidor que soporte PHP (por ejemplo: XAMPP, LAMP, MAMP).
2. Colocar los archivos en el directorio de proyecto de tu servidor local.
3. Abrir los ejercicios desde un navegador accediendo al servidor local (`localhost`). :contentReference[oaicite:1]{index=1}

Opcionalmente, si trabajas con entornos más avanzados, puedes usar contenedores de desarrollo como GitHub Codespaces o Docker para levantar un entorno reproducible. :contentReference[oaicite:2]{index=2}
